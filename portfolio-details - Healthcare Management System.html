Java Servlet : Health Care System

Project information

  • Category: Web Application
  • Project Name: Health Care System
  • Description: A Health Care System is a comprehensive web-based healthcare application designed to bridge the gap between doctors and patients. With three distinct login portals for administrators, doctors, and patients, the platform offers seamless interaction and healthcare management.
  • Developed a web-based health care application featuring three login portals: admin, doctor, and patient.
  • Admin portal for managing doctor and patient registrations, as well as verification processes.
  • Facilitated one-to-one interaction between patients and doctors on an independent, platform-agnostic system, not limited to a specific hospital.
  • Utilized Java Servlet for the backend, handling data processing, user authentication, and session management.
  • Created and maintained the MySQL database to store user information, medical records, and interactions between patients and doctors
